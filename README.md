# FIN-TECH-WEB-APPLICATION

A modern FinTech web application designed to provide secure digital banking services. The system supports user authentication, transaction management, profile handling, and insurance-related features. It is built using a responsive React frontend, a scalable Node.js and Express backend, and MongoDB for persistent data storage.

## Features

- Secure user authentication and authorization  
- User profile creation and management  
- Transaction creation and complete transaction history  
- Insurance and investment-related modules  
- Responsive and user-friendly interface  
- Modular backend architecture for scalability  

---

## Technology Stack

- **Frontend:** React, Tailwind CSS, Recharts  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB (Mongoose ODM)  
- **Other Tools:** Axios, React Router, CORS
